What’s already good vs what we’ll improve

The foundation is solid. The gaps below are the highest-impact items for the next phase of work.

Already working well

  • Secure website (HTTPS) with correct redirects
  • Clear page title and description mentioning De Pijp / Amsterdam
  • Nice preview when sharing on social apps (image + text)
  • Name, address, and phone match across the page
  • Google Maps link points to the real Angelo place
  • Seven languages available (including Arabic RTL)
  • Fast, stable mobile experience (LCP 2.1s, no layout jump)

Priority improvements

  • P0No robots.txt — Google’s “visitor instructions” file is missing
  • P0No sitemap.xml — no clear map of all language pages
  • P0Languages not formally linked in the page head (hreflang)
  • P1Social links are placeholders, not real business profiles
  • P1Opening hours & price range missing from structured business data
  • P1Automatic language redirect may confuse search crawlers

Plain-language guide to the six scores

Each area answers a simple question about how easy it is for Google — and for guests — to find the lounge.

Recommended next steps (in order)

These are the planned improvements for the SEO phase — not yet implemented on this baseline date.

  1. Add robots.txt and a sitemap listing all 7 language home pages
  2. Add proper language tags in the page head so Google knows each translation
  3. Replace placeholder social links with real profiles (or remove them)
  4. Publish opening hours and price range in the business data Google reads
  5. Stop automatic language redirects from confusing search bots
  6. Fill missing image descriptions; add a click-to-call phone link
  7. Connect Google Search Console and submit the sitemap for measurement
